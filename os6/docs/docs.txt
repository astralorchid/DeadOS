readProgram:
    ah = 0 or 1
        0: read head 0
        1: inc through heads
    bx = sector offset
    dh = head
    -Loads sector at 0:1000

pdt.isProgram:
    -Checks if the loaded sector at 0:1000 is a program
    -outputs bx = 0 or 1
        0: no program
        1: program


0x0000:0x0500 - Program Descriptor Table:

+:0x0000 - Start sector (byte)
+:0x0001 - Head (byte)
+:0x0002 - Program name (string byte)
+:0x000A - End program name
